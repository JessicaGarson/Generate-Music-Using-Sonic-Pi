What is Sonic Pi?

Sonic Pi is an IDE that allows us to interact with our computer as if we were using a synth.

Where can I get this?

You can download Sonic Pi at https://sonic-pi.net/

The documenation is in the IDE for you

I got started by just changing the examples.

Let's go on a tour of the IDE

We'll go through and see what the IDE looks like.

Basic syntax of a loop


            live_loop :name_your_loop do
              synth :name_of_synth
              sleep 0.25
            end

Enough talk let's rock


            live_loop :lets_start do
              synth :pretty_bell
              sleep 0.25
            end

Let's add in a sample


            live_loop :second_loop do
              sample :loop_safari
              sleep 0.75
            end

Let's put some bass into the same loop


          live_loop :second_loop do
            sample :loop_safari
            sample :bass_hit_c
            sleep 0.75
          end
